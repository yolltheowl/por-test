# Pappus PoR Probe Target

This is a public test repository used to verify a local agent publication flow.

The probe may:

- fork this repository into a test GitHub account;
- create a temporary branch in the fork;
- write a small probe file under `.pappus-probe/`;
- open a pull request back to this repository.

This repository is intentionally minimal and must not contain private data,
production configuration, credentials, or internal design notes.

Pull requests generated by automated probes are safe to close after validation.
